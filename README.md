# 🎓 Distance Degree Decoded Presentation Reader

A premium PDF presentation and annotation tool designed specifically for educators, trainers, and content creators. It combines powerful presentation features such as auto-scroll, laser pointer with motion trail, advanced annotation tools, and a professional classroom-focused interface.

---

## ✨ Features

### 📄 PDF Viewing
- Upload and view PDF documents
- Full-screen presentation mode
- Zoom In / Zoom Out
- Page navigation (Next, Previous, Jump to Page)
- Thumbnail sidebar for quick navigation

### ⏩ Auto Scroll
- Start, Pause, Resume, and Stop auto-scroll
- Adjustable speed slider
- Smooth scrolling for hands-free presentations

### 🔴 Laser Pointer
- Realistic laser pointer that follows the mouse
- Motion trail effect for better visibility
- Customizable size and color

### ✍️ Annotation Tools
- Freehand Pen
- Highlighter
- Line Pen for perfectly straight lines
- Shapes (Rectangle, Circle, Arrow)
- Text Notes
- Undo and Redo support
- Clear annotations

### 🎨 Professional Interface
- Clean and modern UI
- Optimized for teaching and presentations
- Distance Degree Decoded branding
